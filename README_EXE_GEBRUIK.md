# ADM EXE - Korte Gebruikershandleiding

## 1) Uitpakken
1. Ontvang het zip-bestand met `ADM_portable`.
2. Pak het zip-bestand volledig uit naar een lokale map (bijv. `C:\Tools\ADM`).
3. Open de uitgepakte map en start `ADM.exe`.

## 2) Eerste keer instellen
1. Open in ADM rechtsboven `Settings`.
2. Bij `Datastruct folder` klik je op `Browse...`.
3. Selecteer de juiste datastruct map.
4. Klik `Save`.
5. Kies `Yes` bij `Re-index now?`.

## 3) Juiste datastruct locatie
De juiste datastruct folder staat in:

- `Operations > ADM`

Kies daar de map die de submappen bevat:
- `BOMS`
- `PDF`
- `STEP`
- `SOP`
- `OVERIG`

## 4) Dagelijks gebruik
- Gebruik de zoekbalk om artikelen te vinden.
- Klik een artikel om BOM en documenten te zien.
- Gebruik de assembly tree voor parent/child navigatie.
