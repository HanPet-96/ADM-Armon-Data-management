ADM release stappen (voorbeeld: van 1.0.0.1 naar 1.0.0.2)

1) Open terminal in deze map:
   D:\Onedrive\OneDrive - Armon Products B.V\Project ADM (Armon Document Management) - Project ADM\TestStructuur ADM\Script\ADM-Armon-Data-management

2) One-click release script (aanbevolen):
   release_one_click.bat 1.0.0.2
   Dit doet automatisch:
   - versienummer updaten
   - tests draaien
   - build maken
   - zip maken: dist\ADM_v1.0.0.2_portable.zip

3) Commit en push:
   - Open GitHub Desktop
   - Controleer wijzigingen
   - Commit message: Release v1.0.0.2
   - Push origin

4) Publiceer GitHub Release:
   - Ga naar: https://github.com/HanPet-96/ADM-Armon-Data-management/releases
   - Draft a new release
   - Tag: v1.0.0.2
   - Title: ADM v1.0.0.2
   - Upload: dist\ADM_v1.0.0.2_portable.zip
   - Publish release

5) Update-check in app:
   - App vergelijkt lokale versie met laatste GitHub release tag
   - Is tag hoger? Dan meldt app dat er een nieuwe versie beschikbaar is
   - Settings -> Check for updates opent releases link bij update
